# Number-guess
Description
This is a fun two-player number-guessing game.
One person thinks of a secret number, and the other person tries to guess the digits of that number using logic and clues.

Player 1 secretly writes or stores a number.
Player 2 enters a guess.
The program checks:
How many digits are correct.
How many digits are in the correct position.
Based on these clues, Player 2 keeps guessing until the full number is correctly identified.
The game ends when the guess exactly matches the secret number.
